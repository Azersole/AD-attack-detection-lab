# Active Directory Attack & Detection Lab

A custom-built Windows Active Directory lab designed to simulate a complete attack chain from an authenticated domain user to full Domain Administrator compromise. The project combines offensive security techniques with defensive monitoring using Sysmon and Wazuh SIEM.

## 📌 Overview

This lab was created to practice:

✔️ Active Directory Enumeration  
✔️ Privilege Escalation  
✔️ Kerberoasting  
✔️ DCSync  
✔️ Pass-the-Hash  
✔️ Detection Engineering with Wazuh  
✔️ Sysmon Event Analysis  
✔️ MITRE ATT&CK Mapping
